- Aula 09

		- PDs não clássicas: exemplos de problemas.

- Aula 10

		- Ponteiros. Lista Ligada. Árvore Binária.

- Aula 11

		- Representação de grafos, matriz de adjacência, lista de adjacência. DFS e BFS.

- Aula 13
